# ⚽ Playmaker FC Attendance Management System

A football academy attendance management system built for coaches to track player attendance, generate monthly reports, and manage multiple training groups.

## 🌐 Live Demo

https://elaronaldo7-debug.github.io/playmaker-fc

## 📋 Features

- ✅ Real-time attendance tracking
- ✅ Multiple training groups (Basic 1, Basic 2, U11, U13, U15, U7, U9)
- ✅ Present / Absent management
- ✅ Monthly attendance reports
- ✅ Attendance percentage calculation
- ✅ Export reports
- ✅ Coach-friendly dashboard
- ✅ Firebase real-time synchronization
- ✅ Mobile-friendly interface

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Firebase

## 📊 Modules

### Attendance
- Mark player attendance
- Group-wise attendance tracking
- Present / Absent summary

### Reports
- Monthly attendance reports
- Attendance percentage analysis
- Group performance statistics

### Admin Features
- Manage player records
- View academy attendance statistics
- Export attendance data

## 🎯 Purpose

This project was developed to help football academy coaches manage attendance efficiently and generate reports without manual record keeping.
